
# UrbanWear Collective – CMP309 Web Programming Assignment

This project contains a single HTML page that displays two different layouts
depending on which CSS stylesheet is linked. The goal of the assignment is to
demonstrate layout control using only HTML and CSS (no JavaScript).

---

## Files Included

- **index.html**  
  The only HTML file in the project. It contains six boxes (A–F), representing
  product categories in the fictional clothing brand *UrbanWear Collective*.

- **styleA.css**  
  Implements **Version A**, a vertical layout using CSS Flexbox.  
  - Six boxes arranged vertically  
  - Centered horizontally and spaced evenly vertically  
  - Boxes are 100×100 px  
  - Alternating background colors  
  - Final box (F) has a different color and a 4px solid border  

- **styleB.css**  
  Implements **Version B**, a horizontal layout.  
  - Boxes A–E are lined up horizontally in the top-left corner  
  - Boxes never wrap (`white-space: nowrap`)  
  - Box F is fixed to the bottom-right corner of the window  
  - Boxes are 100×150 px, with a 10px dotted border on the left  
  - Hover effect: yellow background, goldenrod text, hand cursor
